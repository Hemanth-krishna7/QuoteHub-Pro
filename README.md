# QuoteHub Pro

QuoteHub Pro is a full-stack quote discovery and management platform that helps users discover, organize, and revisit inspirational quotes through a modern web experience.

Users can generate quotes in real time, save favorites, create custom collections, track activity, explore analytics, and export beautifully designed quote cards for sharing.

## Live Demo

🔗 https://quotehub-pro.onrender.com

## Features

* Random Quote Generator
* Daily Quote of the Day
* Favorites Management
* Custom Quote Collections
* Quote History Tracking
* Search Functionality
* Activity Timeline
* Analytics Dashboard
* PNG Quote Card Export
* Responsive Light & Dark Mode Interface

## Tech Stack

| Layer           | Technologies                       |
| --------------- | ---------------------------------- |
| Backend         | Python, Flask                      |
| Frontend        | HTML, CSS, JavaScript, Bootstrap 5 |
| Database        | SQLite                             |
| API Integration | ZenQuotes API                      |
| Deployment      | Render                             |
| Version Control | Git & GitHub                       |
